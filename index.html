Web Apps

Web apps are usually built using web development technologies like HTML, CSS, JavaScript, and server-side languages like PHP, Ruby, or Python. They can be accessed from any device with a web browser, making them convenient and widely available.

Examples

            1. Social media platforms (e.g., Facebook, Twitter)
            2. Online banking and e-commerce sites (e.g., Amazon, PayPal)
            3. Email services (e.g., Gmail, Outlook)
            4. Online productivity tools (e.g., Google Docs, Trello)
            5. Educational platforms (e.g., online courses, learning management systems)
            6. News and media websites (e.g., online newspapers, blogs)
            7. Online forums and communities
            8. Web-based games

Types of Webapps

There are two types of Webapps 
            1. Static web apps: Display fixed content, like a brochure website.
            2. Dynamic web apps: Interactive and responsive, like a social media platform.
